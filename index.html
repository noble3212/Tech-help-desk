Blue Screen of Death (BSOD)
Note the stop code. Common ones like MEMORY_MANAGEMENT or IRQL_NOT_LESS_OR_EQUAL often point to RAM or driver issues.
Windows update stuck
Reboot into Safe Mode, run Windows Update troubleshooter, or reset Windows Update components via command line.

üêß Linux Troubles

No WiFi after install
Use `lspci` or `lsusb` to identify the WiFi chip. Install missing drivers via terminal or proprietary firmware via Software & Updates.
"Permission denied" errors
Use `sudo` for administrative tasks. Check file permissions with `ls -l` and change them with `chmod` or `chown` if needed.

üí° General Help

What is BIOS/UEFI?
BIOS/UEFI is firmware that initializes hardware before booting the OS. You can access it by pressing a key like DEL or F2 on startup.
How to clean a PC safely?
Turn off and unplug the PC. Use compressed air for fans, heatsinks, and power supply. Avoid using a vacuum cleaner directly.
