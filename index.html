Overheating CPU
1. Check thermal paste coverage. Clean off old paste with isopropyl alcohol and reapply.
2. Ensure heatsink is properly seated and tightened.
3. Case airflow: add intake/exhaust fans, check fan direction.
4. Use HWMonitor or CoreTemp to check temps under load.
5. Replace thermal paste every 1‚Äì4 years depending on use and environment.

No display but PC powers on
1. Reseat the GPU or try integrated graphics.
2. Try another monitor or cable (HDMI/DP).
3. Reset BIOS (CMOS).
4. Listen for beep codes if motherboard has speaker.
5. Try 1 stick of RAM or swapping RAM slots.

Boot device not found
1. Check BIOS boot order.
2. Confirm drive is detected in BIOS.
3. Try a different SATA port/cable or M.2 slot.
4. Reinstall OS if drive is corrupted.

Blue Screen of Death (BSOD)
1. Note the stop code (e.g., MEMORY_MANAGEMENT, IRQL_NOT_LESS_OR_EQUAL).
2. Run chkdsk /f, sfc /scannow, and Memory Diagnostic Tool.
3. Use dism /online /cleanup-image /restorehealth.
4. Update or roll back drivers (especially GPU, chipset, storage).
5. Kernel anti-cheat (used by games) can cause BSODs‚Äîtest by uninstalling.

Windows update stuck
1. Open Settings > Update & Security > Troubleshoot > Windows Update.
2. Or manually reset update services via CMD:
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
Delete contents of C:\Windows\SoftwareDistribution, then:
net start wuauserv ...etc.
3. Microsoft usually retries failed updates within days.

Windows permission denied
1. Right-click > Run as Administrator.
2. Check folder permissions under Properties > Security tab.
3. Use icacls in CMD to modify permissions if needed.
4. Disable 3rd-party antivirus and test again.

üêß Linux Troubles

No WiFi after install
1. Identify your WiFi chipset: run lspci or lsusb.
2. Use rfkill list to check if the WiFi is blocked.
3. Enable proprietary drivers (e.g., Broadcom) via sudo ubuntu-drivers devices or GUI (Software & Updates > Additional Drivers).
4. If drivers aren't working, try: sudo apt install bcmwl-kernel-source (Broadcom), or relevant package.
üì∫ YouTube: Fix WiFi not working in Linux
"Permission denied" errors
1. Use sudo before commands needing root access.
2. Check file permissions with ls -l. For example:
-rw-r--r-- 1 user user 1234 file.txt
3. Modify permissions: chmod +x script.sh to make executable.
4. Change ownership: sudo chown youruser:youruser file.txt
5. For folders, use -R to apply changes recursively.

Black screen after boot
1. Press Ctrl + Alt + F2 to enter a terminal (TTY2).
2. Try restarting your display manager:
sudo systemctl restart gdm (GNOME)
sudo systemctl restart sddm (KDE)
sudo systemctl restart lightdm (XFCE, others)
3. If using NVIDIA, reinstall or switch to the open-source driver:
sudo apt install nvidia-driver-XXX or nouveau.
üì∫ YouTube: Fix Linux black screen boot
Can't mount external drive
1. Run lsblk or sudo fdisk -l to locate the drive.
2. Mount manually:
sudo mount /dev/sdX1 /mnt
3. If NTFS/FAT32 fails, install:
sudo apt install ntfs-3g
4. For permissions issues:
sudo chown youruser:youruser /mnt

Package manager errors (APT)
1. Broken packages:
sudo apt --fix-broken install
2. Lock file stuck:
sudo rm /var/lib/dpkg/lock* and then:
sudo dpkg --configure -a
3. Clean caches:
sudo apt clean and sudo apt autoremove
4. Update package list:
sudo apt update && sudo apt upgrade

Audio not working
1. Run pavucontrol (install if missing) to check audio routing.
2. Restart audio service:
systemctl --user restart pulseaudio or pipewire
3. If HDMI audio is missing, use alsamixer to check channels.
4. Ensure your user is in the audio group:
sudo usermod -aG audio $USER

What is BIOS/UEFI?
BIOS/UEFI is firmware that initializes hardware and hands off to the OS. Access via DEL, F2, or ESC during startup.
UEFI is newer and supports mouse input, GPT drives, and Secure Boot.

How to clean a PC safely?
1. Turn off and unplug your PC. Press power button to discharge power.
2. Use compressed air on fans, heatsinks, PSU grill, and case vents.
3. Clean filters and fan blades with a soft brush.
4. Avoid static: use an anti-static wrist strap or touch metal frequently.
